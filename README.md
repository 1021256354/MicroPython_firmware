# MicroPython_firmware
MicroPython firmware for Various development board can run micropython.

### STMicroelectronics Boards

* STM32F429DISC
* STM32F7DISC
* NUCLEO_L476RG
* NUCLEO_F401RE
* NUCLEO_F411RE

### TEXAS INSTRUMENTS Boards

* CC3200-LAUNCHXL
* MSP-EXP432P401R

### Third party boards

* PYB Nano V11
* ESP-MP-01
* CANNON
* Neutron
* XMC4700 Relax Kit
