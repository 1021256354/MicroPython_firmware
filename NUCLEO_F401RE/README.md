# Nucleo-F401RE

STMicroelectronics Nucleo-F401RE baord.

Modify I2C1 from PB6/PB7 to PB8/PB9, in order to compatible with Arduino Uno interface.

You may using STM32 ST-LINK Utility to download firmware (HEX).
